Blockchain Simulator

- Edit the Data field to change block content.
- The Nonce is the number you can change to manually find a valid hash.
- Click Mine to instantly find a valid hash starting with "0000".
- The Hash is generated using the Data, Nonce, and Previous Hash for the block.
- Blocks are chained using the previous block's hash. Try modifying a block and see all subsequent blocks become invalid.
